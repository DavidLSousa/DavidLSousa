## Projetos

**[Xpo-cli](https://xpo-cli.vercel.app)** - Opensource CLI
- Criação de uma CLI para abstrair a complexidade de builds mobile, substituindo fluxos baseados em Makefiles por uma ferramenta Node.js moderna e tipada. Desenvolvimento de um sistema de tratamento de erros para credenciais Apple/Google durante o processo de build, evitando falhas silenciosas no pipeline.

**[Xpo Launchpad](https://xpo-launchpad.vercel.app)** - Opensource
- Arquitetura base para React Native (Expo) com foco em escalabilidade. DDD, design tokens (dark/light e multi idiomas), React Query, Axios, mock simples e centralizado sem dependências externas, suporte offline e tooling (Husky, ESLint, Jest).

**[Vixt Framework](https://vixt-framework.vercel.app)** - Opensource
- Micro-framework Vanilla TypeScript de alto desempenho com arquitetura modular e hardening de segurança integrado.
- Disponível publicamente no NPM (`@vixt-framework/core`).

**[imovSystem](https://github.com/orgs/imovSystem/repositories)**
- SAAS para imobiliárias e corretores, com interação com Canal Pro, gerenciamento financeiro e CRM para leads.

**[XPOSystem](https://github.com/orgs/xposystem/repositories)**
- SAAS para cardápio digital ou vitríne de profissionais e produtos, com uma custumização frontend de versátil, sistema de agendamento e financeiro básico.

**[The Flash Cards](https://github.com/DavidLSousa/theflashcards)**
- App de estudo em React Native com criação de cartões de pergunta e resposta, simulados por categoria e dashboard de estatísticas.
